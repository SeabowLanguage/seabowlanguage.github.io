Seabow data types

List of seabow data types:

Type Description Examples
byte A signed 8-bit integer -128 ⟷ 127
ubyte An unsigned 8-bit integer 0 ⟷ 255
short A signed 16-bit integer -32 768 ⟷ 32 767
ushort An unsigned 16-bit integer 0 ⟷ 65 535
int A signed 32-bit integer -2 147 483 648 ⟷ 2 147 483 647
uint An unsigned 32-bit integer 0 ⟷ 4 294 967 295
long A signed 64-bit integer -9 223 372 036 854 775 808 ⟷ 9 223 372 036 854 775 807
ulong An unsigned 64-bit integer 0 ⟷ 18 446 744 073 709 551 615
int128 A signed 128-bit integer ≈ -1.701 x 10³⁸ ⟷ ≈ 1.701 x 10³⁸
uint128 An unsigned 128-bit integer 0 ⟷ ≈ 3.403 x 10³⁸
float A 32-bit decimal number 1.175494 x 10⁻³⁸ ⟷ 3.402824 x 10³⁸ with 6 decimal digits
double A 64-bit decimal number 2.225073858507201 x 10⁻³⁰⁸ ⟷ 1.797693134862316 x 10³⁰⁸ with 15 decimal digits
ldouble A 128-bit decimal number 3.3621 x 10⁻⁴⁹³² ⟷ 1.18973 x 10⁴⁹³² with 18 decimal digits
bool A 8-bit boolean value true or false
char A UTF-32 (32-bit) character All existing character expected non UTF-32 specific character
string A chain of multiple UTF-8 (8-bit) characters Can store all existing characters
array A non-mutable list of seabow values Can store all type of seabow values
list A mutable list of seabow values Can store all type of seabow values
dict A key-value mutable list of seabow values Keys and values can be of all type of seabow values

Special data types:

Special type Description Examples
reference A reference to a seabow variable, used when you want to pass the variable to a function and not a copy of it reference: int&, any&, ...
get ref: &variable
function A block of multiple seabow statements All builtin-functions or user-defined functions
error A seabow execution error ZeroDivisionError ConversionError
error("NewError", "Description")
null A non-initialized value that can be set to any nullable types null
not-null A non-nullable value that can not be set to null Just add a '!' after a type
int!, any!, ...
not-null reference A reference to a non-nullable value Add a '!' and a '&' after a type
int&!, any!&, ...
structures An organization of seabow values that allow user to create a new type Any organization of seabow values
enumerations An enumeration of multiple words that allow user to create a new type that stored this multiple words Any enumeration of words
class / objects An organization of seabow values with functionalities and more that allow user to create a new type Any organization of seabow values with functionalities and more
