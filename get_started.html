Seabow data types

Basic data-types

In seabow, there are many basic types that can be used to store data.

type description examples

ubyte Store an unsigned 8-bit integer 0 ⇿ 255

ushort Store an unsigned 16-bit integer 0 ⇿ 65 535

Special data-types

Seabow has a few special data-types such as:

  • null ⇰ An empty value / nothing
  • any ⇰ Can store all types of data
  • reference ($) ⇰ Store the reference to another data
  • notnullable (!) ⇰ Store data that can not be set to null
  • function ⇰ Store data that represents a seabow function
