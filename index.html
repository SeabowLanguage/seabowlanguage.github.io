What is seabow ?

Seabow is a programming language written in C++ which provides much more readable syntax. It includes an interpreter, a bytecode generator, a bytecode runner and a native compiler in only one command 'seabow'. It is created to provide a brand new language who can be written in one Operating System and run in multiple other Operating Systems. Seabow can manage compile-time and runtime errors, macros, non-nullable values and much more.

A simple look at seabow main function

'main' function is the entry point of a seabow code and is mandatory to execute it.

// Main entry point of a seabow program
void  
main
(
args
) {
    // You can write multiple statements here
    print
(
"Hello, World!"
)
}

In seabow, each statement can be separate by a new-line or a ';'.

Note that the only acceptable argument for main function is 'args'.

How to install seabow ?

You can install seabow by different ways:

  • standard => install seabow with all standard libraries.
  • basic => install seabow without any standard libraries (each can be install separatly).
